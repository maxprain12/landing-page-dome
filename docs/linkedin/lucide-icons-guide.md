## ğŸ’¡ Best Practices

### Color
```css
/* Light backgrounds */
color: #596037;  /* Many green */

/* Dark backgrounds */
color: #E0EAB4;  /* Lighter Many green */

/* Interactive elements */
color: #1A1A1A;  /* Dark for emphasis */
```

### Consistency
- Always use 2px stroke width (except hero at 1.5px)
- Keep round linecap and linejoin
- Use viewBox="0 0 24 24" always
- Color via `currentColor` for flexibility

### Don't
- âŒ Mix icon styles (all Lucide or none)
- âŒ Use filled icons
- âŒ Vary stroke width within same post
- âŒ Scale below 20px (illegible)
- âŒ Use decorative icons (only functional)
